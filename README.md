
# Maharishi S - Legal Portfolio Website

This is a professional resume upload and portfolio website created for hosting on GitHub Pages.

## 🌐 Project Overview

This website includes:

- Professional landing page
- Resume upload (PDF preview feature)
- Responsive modern design
- Clean legal portfolio layout
- Ready for GitHub Pages deployment

---

## 📁 Project Files

- `index.html` – Home page
- `upload.html` – Resume upload page
- `style.css` – Website styling
- `script.js` – PDF preview functionality

---

## 🚀 How to Deploy on GitHub Pages

1. Create a new repository on GitHub.
2. Upload all project files:
   - index.html
   - upload.html
   - style.css
   - script.js
   - README.md
3. Go to **Settings → Pages**
4. Under "Source":
   - Select Branch: `main`
   - Folder: `/root`
5. Click **Save**

Your website will be live within 1–2 minutes.

---

## 💻 How It Works

- The homepage introduces the portfolio.
- The upload page allows users to upload a PDF resume.
- JavaScript dynamically previews the uploaded PDF using an iframe.

⚠ Note: This version only previews the resume locally in the browser.
It does NOT permanently store uploaded files (GitHub Pages does not support backend storage).
